Vos données personnelles

Chez Cookwi, nous pensons que vos données sont sacrées. C'est en gardant cela à l'esprit que nous construisons Cookwi. C'est aussi pour cette raison que nous souhaitons vous donner le plus de détails possible sur ce que nous faisons de vos données : lesquelles nous collectons (le moins possible à vrai dire), comment nous les stockons, et comment nous les utilisons.

De manière générale, aucune de vos données n'est utilisée en dehors du cadre de Cookwi et de son bon fonctionnement. Nous ne traitons pas vos données en dehors des systèmes de Cookwi. Nous ne vendons, et ne vendrons jamais vos données.

Statistiques d'utilisation

Données : Nous sommes suceptible de collecter des informations anonymes (aucune information ne permet de remonter à vous ou votre ordinateur) sur la manière dont vous parcourez les pages du présent site internet. Par exemple, nous pouvons collecter l'ordre des pages dans lequel vous parcourez le site, ou encore le nombre de fois où vous avez visité telle ou telle page.

Utilisation : Nous avons parfois besoin de comprendre comment vous intéragissez avec le site internet afin d'en améliorer l'ergonomie, tout simplement. Ce sont des données précieuses pour nous. Par exemple, grâces à ces données, nous pourrions identifier que personne ne clique jamais sur un bouton car il n'est pas assez visible. Nous pourrions alors décider d'en changer la couleur en conséquence.

Stockage : Nous stockons ces données dans une base de donnée privée à Cookwi et sécurisée. De plus, comme déjà évoqué, ces données ne vous identifient pas. Nous ne collectons pas votre adresse IP, ni aucune information personnelle.

Nom et prénom

Utilisation : Nous utilisons votre nom et votre prénom afin de les afficher dans votre profil ou sur vos recettes. Ils permettent d'identifier à qui appartiennent des recettes. Sans cela, vous auriez un numéro d'identification unique "technique" peu compréhensible par un être humain - quelque chose dans le genre : "5b78cf34ae". Pas très pratique.

Stockage : Nous stockons ces informations dans une base de donnée privée et sécurisée.

Adresse éléctronique

Utilisation : Nous collectons votre adresse e-mail afin d'avoir la possibilité de vous joindre dans le cadre de votre compte Cookwi. Cela nous permet par exemple de vous envoyer des informations importantes sur la sécurité de votre compte, ou encore sur des évolutions majeures du site internet.

Stockage : Nous stockons cette information dans une base de donnée privée et sécurisée.

Recettes

Utilisation : Vos recettes vous appartiennent. Nous les utilisons afin de vous les montrer, et les montrer aux personnes avec qui vous avez décidé de les partager. Nous ne publierons jamais vos recettes sans votre consentement explicite préalable (il se peut qu'un jour nous demendions gentillement).

Stockage : Nous stockons vos recettes dans une base de données privée et sécurisée.

Photos de recettes

Utilisation : Les photos de vos recettes sont utilisées afin de vous montrer vos recettes illustrées. Elles sont susceptible d'être également affichées aux personnes avec qui vous avez partagé vos recettes.

Stockage : Les images et photographies sont stockées sur un serveur de type S3 privé (nous n'utilisons pas de cloud).
